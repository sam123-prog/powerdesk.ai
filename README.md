# powerdesk.ai
author- sam
Digital helpdesk and ticket management system for submitting, tracking, and organizing user queries via a secure web interface. Data is stored locally to ensure privacy. Built for adaptability, with future plans for backend integration and AI-powered chatbot features. Suitable for multiple organizational use cases.
Features
Submit helpdesk queries/messages from an intuitive frontend

View history of queries/messages (local storage)

User guidelines provided for smooth onboarding

Everything operates offline (browser-side only for now)

Modular HTML files for easy updates and expansion

Tech Stack
Frontend: HTML, CSS, JavaScript

Storage: Browser localStorage (Planned: Python, Flask, SQLite, CSV export)

Planned: AI chatbot, backend integration
